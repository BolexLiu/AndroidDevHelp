# AndroidDevHelp
该工程用来解决android同学编程中疑难杂症的辅助仓库。灾难性的bug问题请发Issues！

  

*注：请勿滥用，遵循理性和标准的格式来发起求助,模版在下方。*

# 如何求助

 点击Issues 创建它。然后将你的问题描述仔细，若有错误代码务必贴上来。
